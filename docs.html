DDN INFINIA RAG Performance Demo

Comparing DDN INFINIA vs AWS S3 storage performance with NVIDIA NeMo AI acceleration

Overview

This demo showcases the performance advantages of DDN INFINIA high-performance storage for enterprise RAG (Retrieval-Augmented Generation) applications. It provides real-time TTFB comparisons between DDN INFINIA and AWS S3, with NVIDIA NeMo AI integration.

Performance Metrics

Metric DDN INFINIA AWS S3 Improvement
Upload TTFB ~20ms ~200ms 90% faster
Retrieval TTFB ~25ms ~250ms 90% faster
End-to-end RAG <100ms 300-500ms 3-5x faster

Quick Start

Prerequisites

  • Python 3.9+
  • Node.js 18+
  • NVIDIA API Key from build.nvidia.com

Installation

# Run installation script
./install.sh

# Or manually:
cd backend && python3 -m venv venv && source venv/bin/activate && pip install -r requirements.txt
cd ../frontend && npm install

Configuration

# Add your NVIDIA API key
cp backend/.env.example backend/.env
# Edit backend/.env and add NVIDIA_API_KEY=nvapi-xxxx

Running

Terminal 1 - Backend:

cd backend
source venv/bin/activate
python main.py
# API: http://localhost:8000
# Docs: http://localhost:8000/docs

Terminal 2 - Frontend:

cd frontend
npm run dev
# App: http://localhost:5173

Features

Storage Performance Comparison

  • Dual Storage — Chunks stored to AWS S3 and DDN INFINIA simultaneously
  • Real-time TTFB — Time-to-First-Byte measurement per operation
  • Win Rate Tracking — Performance comparison statistics
  • Live Metrics — Upload speed, retrieval speed, improvement percentages

NVIDIA NeMo AI Integration

  • Nemotron 70B LLM — Enterprise-grade language model
  • NeMo Reranker — 20-30% relevance improvement
  • Content Guardrails — Input/output safety filtering
  • Multi-model Support — Nemotron, Llama 3.1, Mixtral

Document Processing

  • Multi-format — PDF, DOCX, XLSX, PPTX, CSV, TXT
  • Smart Chunking — Semantic text splitting
  • Deduplication — Prevents duplicate chunks
  • Metadata Enrichment — Source tracking, timestamps

Business Impact Analysis

  • ROI Calculator — Enterprise-scale financial modeling
  • 9 Parameters — Queries, users, chunks, TTFB, GPU costs
  • Detailed Reports — GPU savings, productivity gains, 5-year projections
